Moire API
=========

Demarrage:
----------

Rouler les commandes :

	sudo npm install
	bower install
	grunt build

Pour lancer le serveurs plusieurs options:

    - Dev :
        grunt debug

    - Prod :
        forever start moireApi

    - Autre :
        nohup node moireApi
        node moireApi
